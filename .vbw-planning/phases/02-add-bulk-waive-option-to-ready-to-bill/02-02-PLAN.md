## Objective

Add client card selection, a "Bulk Waive" action button, and a confirmation modal to the Ready to Bill page (`UnbilledClientsSection`). When confirmed, the UI calls the bulk-waive API endpoint and refreshes the client list.

### Task 4: Wire up bulk waive API call and refresh

**File:** `app/src/components/billing/UnbilledClientsSection.tsx` (MODIFY -- same file as Task 1)

Integrate the modal and API call:

1. **Import and render** `BulkWaiveConfirmModal` with appropriate props.

2. **Prepare modal data:** When "Bulk Waive" button is clicked, build the `selectedClients` array from the current `clients` state (match by id from `selectedClientIds`) to pass name, hours, entry count to the modal.

3. **`handleBulkWaive` function:**
   ```tsx
   const handleBulkWaive = async () => {
     setIsWaiving(true);
     try {
       const res = await fetch("/api/timesheets/bulk-waive", {
         method: "PATCH",
         headers: { "Content-Type": "application/json" },
         body: JSON.stringify({
           clientIds: Array.from(selectedClientIds),
           ...(dateRange.from && { dateFrom: dateRange.from }),
           ...(dateRange.to && { dateTo: dateRange.to }),
         }),
       });
       if (!res.ok) throw new Error("Failed to waive entries");
       const data = await res.json();
       // Clear selection and close modal
       setSelectedClientIds(new Set());
       setShowConfirmModal(false);
       // Refetch unbilled clients to remove waived entries from view
       // Use existing fetch mechanism (the component already has a fetch function)
     } catch (error) {
       console.error("Bulk waive error:", error);
       // Could show error state -- but keep simple for MVP
     } finally {
       setIsWaiving(false);
     }
   };
   ```

4. **Refetch after waive:** Call the existing data-fetching function that loads unbilled clients (the component already has this for initial load and filter changes). Waived entries will automatically disappear since the unbilled-summary API excludes `isWrittenOff = true`.
